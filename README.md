# Disease Prediction System

A web-based symptom checker built for the K J Somaiya School of Engineering.

## Features
- 56 symptoms across 7 categories
- Predicts from 20 common diseases
- Match percentage scoring
- Login & Register system

## How to Run
Just open `login.html` in any browser. No server needed.
